Centos7系统离线安装nginx步骤(亲测，超详细步骤)

  • 首页
  • 新闻资讯
  • 正文

1.整理离线安装需要的包，打包成nginxOfflinePack.tar.gz 压缩包复制到centos相应目录

我整理的包已上传到csdn资源库 ，下载地址

https://download.csdn.net/download/qq445829096/12451908

2.下载完成后复制到centos相应目录，推荐一个linux远程神器MobaXterm

执行命令，文件复制功能都很好用，还有小游戏...

3.解压nginxOfflinePack.tar.gz  ，我这边是解压到/dev/nginxOfflinePack 目录

cd到存放nginxOfflinePack.tar.gz的目录

执行tar zxvf nginxOfflinePack.tar.gz

4.

安装
开始安装

把包放在一个文件夹中，cd到nginxOfflinePack目录

然后运行命令：rpm -Uvh *.rpm --nodeps --force


安装pcre：
nginxOfflinePack目录 执行

tar -zxvf pcre-8.43.tar.gz

之后cd到pcre-8.43目录 依次执行以下3个指令安装
./configure
make
make install

安装openssl：

nginxOfflinePack目录 执行
tar -zxvf openssl-1.0.2r.tar.gz
之后cd到openssl-1.0.2r目录 依次执行以下3个指令安装


./configure或者./config  (如果./configure提示上图错误，则使用./config)
make
make install

安装zlib：

nginxOfflinePack目录 执行
tar -zxvf zlib-1.2.11.tar.gz

之后cd到zlib-1.2.11目录 依次执行以下3个指令安装
./configure
make
make install


安装nginx：

nginxOfflinePack目录 执行
tar -zxvf nginx-1.16.0.tar.gz

之后cd到nginx-1.16.0目录 依次执行以下3个指令安装
./configure
make
make install

注意 执行make install后 nginx的安装目录会打印出来，不指定的话默认都是该目录

初次启动
cd /usr/local/nginx/sbin
执行指令 ./nginx

./表示是当前目录
浏览器已可访问nginx，打开浏览器访问此机器的 IP，如果浏览器出现 Welcome to nginx! 则表示 Nginx 安装并运行成功。；

如果暂没有window电脑可以访问到也可以执行指令查看

任意路径执行执行 curl http://127.0.0.1   (nginx配置文件配置的默认端口是80)

停止nginx

cd /usr/local/nginx/sbin

执行指令./nginx -s stop

重启nginx

cd /usr/local/nginx/sbin

执行指令./nginx -s reload

查看nginx是够启动

ps -ef|grep nginx
