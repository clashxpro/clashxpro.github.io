第一章：认识小火箭 Shadowrocket

1.1 什么是小火箭？

小火箭 Shadowrocket 是一款专为 iOS 设备设计的网络代理客户端，支持 Shadowsocks、Vmess、Trojan 等多种主流代理协议。它不仅能帮助你访问被限制的网站和服务，还能有效保护你的网络隐私，防止数据被窃取或监控。

1.2 小火箭的核心优势

  • 用户友好：界面简洁直观，新手也能快速上手。
  • 多协议支持：兼容 Shadowsocks、Vmess、Trojan 等多种协议，适应不同网络环境。
  • 灵活配置：支持自定义 DNS、规则设置和代理策略，满足个性化需求。
  • 隐私保护：通过加密传输数据，防止 ISP 或第三方监控你的网络活动。

第二章：下载与安装

2.1 官方渠道获取

小火箭 Shadowrocket 在 App Store 中提供下载，但由于某些地区的限制，可能需要切换 Apple ID 地区或通过其他方式获取。

步骤：
1. 打开 App Store，切换至支持 Shadowrocket 的地区（如美区、港区）。
2. 搜索 "Shadowrocket" 并购买（通常售价 $2.99）。
3. 下载并安装。

2.2 替代下载方式

如果无法通过 App Store 获取，可尝试以下方法：
- 通过第三方签名工具（如 AltStore）安装 IPA 文件。

⚠️ 注意：非官方渠道可能存在安全风险，建议优先选择正版。

第三章：基础配置教程

3.1 添加服务器

  1. 打开 Shadowrocket，点击右上角 + 按钮。
  2. 选择代理类型（如 Shadowsocks、Vmess）。
  3. 填写服务器信息（地址、端口、密码、加密方式等）。
  4. 保存并返回主界面。

3.2 连接与断开

  • 连接：点击服务器名称，再点击底部 连接 按钮。
  • 断开：再次点击 连接 按钮即可。

3.3 检查连接状态

  • 成功连接后，状态栏会显示 VPN 图标。
  • 可在 Shadowrocket 主界面查看实时流量和连接时长。

第四章：高级功能与优化

4.1 自定义 DNS

优化 DNS 可以提升解析速度并避免污染：
1. 进入 设置 → DNS。
2. 添加公共 DNS（如 1.1.1.1、8.8.8.8）。
3. 启用 DNS over HTTPS (DoH) 增强隐私。

4.2 规则配置

通过规则可以实现智能分流（如国内直连、国外走代理）：
1. 进入 配置 → 规则。
2. 添加规则集（如 GFWList、自定义规则）。
3. 设置匹配策略（DIRECT、PROXY、REJECT）。

4.3 日志与调试

如果遇到连接问题，可查看日志排查：
1. 进入 日志 选项卡。
2. 检查错误信息（如超时、认证失败）。
3. 根据日志调整服务器或网络设置。

第五章：常见问题与解决方案

5.1 连接失败怎么办？

  • 检查服务器信息：确保地址、端口、密码正确。
  • 更换协议：尝试 Shadowsocks 或 Vmess 不同加密方式。
  • 网络环境：某些 WiFi 可能屏蔽代理，可切换 4G/5G 测试。

5.2 如何提升速度？

  • 选择低延迟的服务器。
  • 启用 UDP 转发（若支持）。
  • 关闭不必要的规则以减少性能开销。

5.3 是否支持多设备？

Shadowrocket 仅限 iOS 设备，但可通过订阅方式同步服务器配置到其他客户端（如 Clash、Surge）。

第六章：安全与隐私建议

  1. 定期更换密码：避免长期使用同一组认证信息。
  2. 启用 TLS 加密：Vmess 协议建议搭配 TLS 以增强安全性。
  3. 避免公共 WiFi：即使使用代理，公共网络仍可能存在风险。

FAQ

Hiddify Next 用户流量配额如何设置？
管理员可为每个用户设置带宽和流量上限，超出限制可自动降速或阻止访问，保证服务器资源合理分配，避免单用户占用过多带宽，提高整体网络公平性和使用效率。
SSR 的混淆插件如何提高抗封锁能力？
通过混淆插件可以模拟正常 HTTPS 或 HTTP 流量，隐藏真实通信特征。合理选择混淆方式，可有效避免 DPI 检测，提高节点可用性和连接稳定性，保证长期科学上网。
Clash 自动节点切换如何改善网页访问速度？
Clash 结合策略组和延迟检测功能，自动选择延迟低、稳定性高的节点处理网页流量。提升访问速度和稳定性，减少加载时间和浏览卡顿，优化整体网络体验。
Surfboard 的节点策略组如何提升使用体验？
策略组可将节点按延迟、用途或地理位置分类，通过规则自动选择最优节点。用户无需手动切换节点即可获得最佳速度和稳定性，提高浏览和观看流媒体的体验。
QuantumultX 如何通过策略组优化多应用分流？
QuantumultX 可以根据应用类型、端口和协议设置策略组，自动选择最优节点处理不同流量。这样可保证网页浏览、视频播放和游戏访问顺畅，提高整体网络效率。
VLESS + XTLS 对高延迟网络有哪些优化？
XTLS 模式减少握手和加密开销，适合高延迟或不稳定网络环境。结合策略组和节点切换，可提高网页浏览、视频播放和游戏的稳定性和流畅度，改善用户体验。
VMess 节点使用 AEAD 加密有何优势？
AEAD 加密能提供更强的安全性和抗重放能力，同时减少加密头泄露特征。相比普通加密，更适合高封锁环境，兼顾安全与性能，保证长期稳定使用。
V2Ray 的 WebSocket 传输与 TCP 有何区别？
WebSocket 通过 HTTP 封装 TCP 流量，适合伪装成正常网页请求，增加隐蔽性。相比原生 TCP，可更好地绕过封锁和 DPI 检测，但略增加协议开销。
Clash 支持哪些协议？
Clash 支持多种主流代理协议，包括 VMess、VLess、Trojan、Shadowsocks、Snell、Socks5 等。通过配置文件或订阅源，用户可以灵活组合不同协议的节点，实现复杂的分流策略与优先级控制。
QuantumultX 分流规则如何优化游戏体验？
QuantumultX 可按端口、协议或应用设置分流规则，将游戏流量走低延迟节点。结合策略组自动切换，可保证游戏连接稳定，减少掉线和延迟，提高流畅度。
